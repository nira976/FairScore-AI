# FairScore AI

Explainable Credit Risk Assessment Platform for Gig Economy Workers using XGBoost and SHAP.

## Features

- Multiclass Credit Risk Prediction
- Explainable AI using SHAP
- Interactive Streamlit Dashboard
- Designed specifically for Gig Economy Workers

## Tech Stack

- Python
- Pandas
- Scikit-Learn
- XGBoost
- SHAP
- Streamlit
